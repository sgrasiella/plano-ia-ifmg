Introdução

A API do Plano de IA do IFMG oferece acesso a modelos ajustados especificamente para o contexto educacional brasileiro. A comunicação é baseada em REST, utilizando representações JSON de transferência.

Gere sua Chave de API

Identifique-se como aluno ou professor para receber sua key gratuita.

Fazendo sua Primeira Requisição

A autenticação é feita via header Authorization: Bearer. O endpoint primário de completions é altamente compatível com o formato da OpenAI, facilitando adaptações de código:

POST https://api.ia.ifmg.edu.br/v1/completions
# Exemplo de requisição em Python usando o pacote requests

import requests
import json

url = "https://api.ia.ifmg.edu.br/v1/completions"
headers = {
    "Authorization": "Bearer SEU_TOKEN_AQUI",
    "Content-Type": "application/json"
}

payload = {
    "model": "ifmg-edu-7b",
    "prompt": "Explique o conceito de redes neurais para um estudante do ensino médio",
    "max_tokens": 250
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
